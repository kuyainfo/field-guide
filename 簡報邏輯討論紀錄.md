# 簡報邏輯討論紀錄

## 核心觀念：辯論三角形

每一頁簡報都應自成一個三角形：
- **主張**：這頁要說的核心觀點（要能被人反對，有立場）
- **材料**：用來支撐主張的內容（數據、案例、圖片、故事）
- **論述**：把材料和主張連起來，讓人信服

整份簡報也是一個更大的三角形：
- **大主張**：客戶看完後應該相信的那一件事
- **材料**：每一頁的小三角形
- **論述**：每頁如何支撐大主張

**關鍵原則：客戶不會幫你腦補。你沒說的，他們就當作不存在。**

---

## 今日最大發現

三份簡報都有同一個問題：**內容都有，但大主張沒有被說出來。**

根本原因是出發點錯誤：
- ❌ 從「我能提供什麼」出發 → 做出服務目錄
- ✅ 從「我要讓客戶相信什麼」出發 → 做出有說服力的提案

三份簡報缺少的東西大同小異：
1. 大主張沒說出來
2. 論述沒有連結材料和主張

**修改公式（三份都適用）：**
1. 第一頁加大主張
2. 每一頁把slogan移到前面當主張
3. 每一頁加一句論述，說明這個材料為什麼能支撐主張

## 一、尾牙主題總覽（酷亞_2026主題總覽v0.2）

### 簡報定位
- 讀者：潛在客戶
- 目的：讓客戶看到主題就心動，想要選它

### 核心問題
- 現有結構是供給清單：場地布置 → 拍照設置 → 迎賓活動 → 延伸設計 → 表演推薦
- 客戶的思維順序是：感覺 → 玩法 → 細節
- 兩者不一致，導致客戶看不到「氛圍」，只看到「清單」

### 大主張（缺少，需加入第一頁）
**「不管你選哪個主題，我們都能讓整晚從頭到尾沉浸其中」**

### 四個主題的獨特差異點

| 主題 | 獨特差異 | 主張方向 |
|------|---------|---------|
| 多巴胺 | 視覺衝擊力最強，最好拍照 | 今晚每一張照片，都是最值得炫耀的畫面 |
| 賽車 | 團隊衝勁、目標感 | 用極速感點燃全員衝勁 |
| 運動 | 多元參與、鼓舞士氣 | 用勝利的氛圍讓每個人都是主角 |
| CASINO | 高奢氛圍、趣味狂歡 | 一秒置身拉斯維加斯的頂級不夜城 |

### 建議結構調整
每個主題改為：**主題主張（一句話）→ 特色（差異點）→ 執行方式**

## 二、家庭日總覽（家庭日總覽.pptx）

### 簡報定位
- 讀者：潛在客戶
- 目的：給沒有想法的客戶一個方向選擇，讓人心動

### 核心問題
- 現有結構像「服務目錄」，不是「讓人心動的提案」
- 大主張沒有出現在簡報裡
- 餐飲選擇放在後面，客戶不知道跟哪個類型搭配

### 大主張（缺少，需加入第一頁）
**「每一種家庭日，都有它最對的樣子」**

### 五種類型的獨特差異點

| 類型 | 獨特差異 | 主張方向 |
|------|---------|---------|
| 戶外風格型 | 真正脫離辦公室，在大自然放鬆 | 脫離辦公室，在大自然裡真正放鬆 |
| 園區包場型 | 結合場地設施的闖關機制與表演 | 把整個園區變成你們公司專屬的冒險場 |
| 闖關競技型 | 用競技點燃團隊默契 | 用勝負慾點燃團隊默契 |
| 室內體驗型 | 舒適精緻、不受天氣影響的深度體驗 | 不用出門吹風曬太陽，也能有最有深度的家庭日體驗 |
| ESG永續型 | 從頭到尾以永續為核心設計，不是貼標籤 | 讓家庭日本身，就是一份遞給員工的ESG成績單 |

### 建議結構調整

**第一頁：大主張**
「每一種家庭日，都有它最對的樣子」

**第二頁以後：五種類型**
每種類型包含——
- 這種類型的主張（一句話）
- 特色（與其他四種的差異）
- 執行方式
- 推薦餐飲配套（小標籤即可）

**餐飲頁面：保留，改為配套說明**

## 三、迎賓道具總覽（酷亞_2026迎賓道具.pptx）

### 簡報定位
- 讀者：已確定要辦尾牙、有迎賓環節需求的潛在客戶
- 目的：讓客戶選到適合的迎賓道具租借方案

### 迎賓道具的本質
迎賓時段是晚宴開始前的緩衝時間，賓客陸續抵達、沒有道具就會乾等、氣氛冷、消磨對晚宴的期待感。

迎賓道具真正的價值不只是消磨時間，而是**為晚宴暖場、製造期待感**。

### 大主張（缺少，需加入第一頁）
**「讓賓客還沒入座，就已經嗨起來了」**

### 六種類型的問題診斷
現有分類名稱（Casino、復古、競技、機台、運動、組裝）是按道具性質分類，不是按客戶的選擇邏輯分類，導致客戶看到名稱沒有畫面。

**客戶選道具的真實邏輯：**
- 有主題 → 按主題選
- 沒主題 → 按員工屬性選（好勝心強選競技、有打牌習慣選Casino、平常少運動選運動類）
- 或是：什麼好玩選什麼

### 每種類型缺少的三角形元素

**現有結構：** 類型名稱（描述）→ slogan（主張）→ 道具項目（材料）

**問題：**
- 主張（slogan）被藏在類型名稱後面，客戶第一眼看不到
- 論述幾乎沒有——只列道具名稱，沒說為什麼這組道具能讓迎賓時段嗨起來

### 各類型論述方向

| 類型 | 論述核心 |
|------|---------|
| Casino | 不需要賭博技巧，純粹靠手氣，任何人都能立刻上手享受勝負快感 |
| 復古 | 跨越世代，年長同仁喚起童年記憶，年輕人感受復古新鮮感，兩種人都能玩得起來 |
| 競技 | 靠實力不靠運氣，賓客自然認真投入，現場氣氛自然熱起來 |
| 機台 | 沉浸式體驗，讓賓客切換至玩家模式，忘記在等待 |
| 運動 | 喚醒肌肉記憶，簡單上手又有成就感 |
| 組裝 | 需要協作完成，自然引發賓客之間的互動與交流 |

### 建議結構調整
**第一頁：大主張**
「讓賓客還沒入座，就已經嗨起來了」

**每個類型頁面改為：**
slogan（主張）→ 論述（為什麼這組道具能做到）→ 道具項目（材料）

---

## 下次做新簡報前，先回答這三個問題

1. **這份簡報的讀者是誰？**
2. **他們看完之後要相信什麼？（大主張）**
3. **我要用什麼來支撐這個主張？（材料）**

寫完這三件事，再開始動手做。
